# Incident App Instructions

To start the Flask application in the correct Conda environment, use the provided shell script:

```bash
bash run_app.sh
```

This script will:
1. Initialize Conda from `/home/apwheele/miniconda3/`.
2. Activate the `inc_app` environment.
3. Run the Flask application via `flask run`.

The application will be available at `http://127.0.0.1:5000`.
